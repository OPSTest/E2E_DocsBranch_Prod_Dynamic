# <a name="make-resources-more-secure-with-azure-advisor"></a>使用 Azure 顾问提高资源的安全性

Azure 顾问针对所有 Azure 资源提供一个一致且统一的建议视图。 它与 Azure 安全中心集成，提供安全性建议。 可在顾问仪表板的“安全”**** 选项卡获取安全性建议。

安全中心有助于预防、检测和响应威胁，同时增加 Azure 资源的可见性和安全可控性。 可定期分析 Azure 资源的安全状态。 安全中心识别到潜在的安全漏洞时，会创建建议。 此建议指导完成配置所需控件的过程。 

有关安全建议的详细信息，请参阅[Azure 安全中心管理安全建议](https://azure.microsoft.com/documentation/articles/security-center-recommendations/)。

## <a name="how-to-access-security-recommendations-in-azure-advisor"></a>如何访问 Azure 顾问中的安全性建议

1. 登录 [Azure 门户](https://portal.azure.com)，并打开[顾问](https://aka.ms/azureadvisordashboard)。

2.  在顾问仪表板中，单击“安全性”**** 选项卡。
